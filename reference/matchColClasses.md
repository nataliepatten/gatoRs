# matchColClasses

This function allows you to match the column classes of two data frames.
This function requires package base. This function does not require user
input. Source:
https://stackoverflow.com/questions/49215193/r-error-cant-join-on-because-of-incompatible-types

## Usage

``` r
matchColClasses(df1, df2)
```

## Arguments

- df1:

  A data frame

- df2:

  A data frame

## Value

df2 with corrected column classes
